## content

### 1. 專案一句話
Aegis Knowledge Core 是一個本機 RAG 知識庫 Demo，整合 LangChain、LangGraph、ChromaDB、HuggingFace Embeddings、Ollama Local LLM 與 Streamlit，支援文件向量化、語意檢索、來源引用、RAG Prompt 展示與 embedding 視覺化。

### 2. 技術亮點條列
- 使用 LangChain 建立文件載入、文字切分、embedding 與 Chroma vector store 流程。
- 使用 LangGraph 將 RAG 查詢拆成 retrieval、context selection、prompt building 與 local LLM generation 節點。
- 使用 ChromaDB 作為本機 persistent vector database。
- 使用 Ollama 與 `langchain-ollama` 在本機生成 RAG 回答。
- 使用 Streamlit 建立可互動 Demo，展示 Answer Mode、Evidence、RAG Prompt 與 Vector Map。

### 3. 專案介紹說法
這個專案不是只呼叫 LLM API，而是做出完整的本機 RAG workflow。資料從 `data/` 匯入後會被切成 chunks，轉成 embeddings 存入 Chroma；查詢時 LangGraph 會管理檢索、整理 context、建立 RAG prompt，最後交給 Ollama 本機 LLM 生成回答並保留來源。

### 4. 可延伸能力
此專案可以延伸成企業內部知識庫、客服 FAQ 助理、技術文件搜尋或專案 onboarding assistant。未來可加入 PDF loader、Notion connector、權限控管、reranker、evaluation dataset 與部署流程。
